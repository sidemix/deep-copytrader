# Polymarket Copytrading Bot

A simple but working copytrading bot for Polymarket that automatically copies trades from leader wallets.

## Features

- Monitor multiple Polymarket wallets
- Automatic trade copying with configurable percentages
- Basic risk management (position sizing, exposure limits)
- Web dashboard for management
- Test/Dry-run mode
- Deployable on Render
